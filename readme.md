# 3. Methods

## **3.1 传统方法**

### 3.1.1 特征表示

- RGB 转换为 HSV 或 Lab
- 提取绿色区域特征

原因：作物在颜色空间中具有明显分布特征

------

### 3.1.2 分割方法

------

### 3.1.3 后处理

- 形态学操作：
  - 开运算
  - 闭运算
- 去除噪声区域

------

### 3.1.4 方法局限性

- 对光照敏感
- 对噪声敏感
- 无法处理复杂背景

------

## **3.2 机器学习方法**

### 3.2.1 特征提取

- 颜色特征（RGB/HSV）
- 纹理特征（LBP）

解释：纹理特征可以区分作物与土壤的结构差异
