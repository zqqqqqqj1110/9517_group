## 8. conclusion

针对传统学习方法，有如下结论

C1:加入ExG/LBP能提升F1与loU

C2:后处理主要提升 precision（去噪）但可能损失 recall

C3:在 noise/blur/dark 下，改进版比基础版更稳

针对深度学习学习方法，有如下结论

“C4 成立/不成立：RF vs XGB 指标变化”

“C5 成立/不成立：平衡策略是否显著提升 recall/IoU”

“C6 成立/不成立：阈值是否导致稳定的 P-R 权衡”

“C7 成立/不成立：鲁棒性退化斜率是否更缓”

针对深度学习学习方法，有如下结论

C1：U-Net 在当前实验中整体优于 DeepLabV3，主指标 F1 / IoU 更高
C2：DeepLabV3 的 precision-recall 权衡可由阈值与训练策略部分调节，但当前最优点仍弱于 U-Net
C3：在 noise / blur / dark / occlusion 扰动下，U-Net 整体更稳；其中 blur 最稳，gaussian noise 退化最明显
